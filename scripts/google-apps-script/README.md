# Google Apps Script: создание таблицы шаблонов

Скрипт создаёт Google Таблицу с **полной строкой заголовков** (все 10 колонок бэкапа) и открывает доступ **«Все, у кого есть ссылка» → Редактор**, чтобы пользователь мог сразу редактировать таблицу.

## Развёртывание

1. Откройте [script.google.com](https://script.google.com), создайте новый проект.
2. Вставьте код из `CreateTemplatesSheet.gs` в файл `Code.gs`.
3. **Секрет:** Файл → Проект → Свойства проекта (или иконка шестерёнки) → Свойства скрипта. Добавьте свойство `SECRET` со значением, совпадающим с `APPS_SCRIPT_SECRET` в `.env` бэкенда.
4. Разверните как веб-приложение: Развернуть → Новое развёртывание → Тип «Веб-приложение»:
   - **Запуск от имени:** Вы (ваш аккаунт)
   - **У кого есть доступ:** Все
5. Скопируйте URL веб-приложения в `APPS_SCRIPT_URL` в `.env` бэкенда.

## Формат запроса (бэкенд)

Бэкенд отправляет POST с телом:

```json
{
  "secret": "<APPS_SCRIPT_SECRET>",
  "userId": "<id пользователя>",
  "name": "WA Templates - <userId>",
  "headers": ["enabled", "order", "title", "text", "media_url", "send_media_as_file", "wa_speed_factor", "tg_speed_factor", "wa_default_send_time", "tg_default_send_time"]
}
```

Если поле `headers` не передано или пустое, скрипт подставит перечисленный выше список по умолчанию.

## Результат

- Таблица создаётся в Google Drive аккаунта, от которого развёрнут скрипт.
- Первая строка заполняется заголовками.
- Доступ: «Все, у кого есть ссылка» могут **редактировать**.
- В ответе возвращаются `editUrl`, `spreadsheetId`, `csvUrl`.
