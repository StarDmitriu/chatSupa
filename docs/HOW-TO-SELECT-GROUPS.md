# Как выбрать группы для рассылки

## Проблема

При попытке запустить кампанию появляется ошибка:
> "Нет выбранных групп для рассылки. Зайдите в раздел групп WA/TG, отметьте нужные группы и попробуйте снова."

## Решение

### Для WhatsApp групп

1. **Откройте раздел групп WhatsApp:**
   - Перейдите в меню: **"Группы"** → **"WhatsApp"**
   - Или откройте: `https://chatrassylka.ru/dashboard/groups`

2. **Синхронизируйте группы (если их нет):**
   - Нажмите кнопку **"Синхронизировать группы"**
   - Убедитесь, что WhatsApp подключён (QR-код отсканирован)

3. **Выберите группы:**
   - Поставьте галочки напротив нужных групп в колонке "Выбрано"
   - Или используйте кнопки **"Выбрать все"** / **"Снять все"**
   - Выбор сохраняется автоматически при клике на галочку

4. **Проверьте выбор:**
   - В верхней части страницы видно количество выбранных групп
   - Например: "Выбрано: 5 · Доступно: 12"

### Для Telegram групп

1. **Откройте раздел групп Telegram:**
   - Перейдите в меню: **"Группы"** → **"Telegram"**
   - Или откройте: `https://chatrassylka.ru/dashboard/telegram-groups`

2. **Синхронизируйте группы (если их нет):**
   - Нажмите кнопку **"Синхронизировать группы"**
   - Убедитесь, что Telegram подключён (код подтверждения введён)

3. **Выберите группы:**
   - Поставьте галочки напротив нужных групп в колонке "Выбрано"
   - Или используйте кнопки **"Выбрать все"** / **"Снять все"**
   - Выбор сохраняется автоматически при клике на галочку

4. **Проверьте выбор:**
   - В верхней части страницы видно количество выбранных групп

## После выбора групп

После того как вы выбрали группы:

1. **Вернитесь в раздел "Кампании"**
2. **Создайте новую кампанию или запустите существующую**
3. **Ошибка "Нет выбранных групп" больше не должна появляться**

## Важно

- **Группы нужно выбирать отдельно для WA и TG** — выбор в одном разделе не влияет на другой
- **Выбор сохраняется автоматически** — не нужно нажимать кнопку "Сохранить"
- **Если групп нет** — сначала синхронизируйте их (кнопка "Синхронизировать группы")
- **Если WhatsApp/Telegram не подключён** — сначала подключите аккаунт в разделе "Личный кабинет"

## Проверка через API

Если нужно проверить выбранные группы через API:

```bash
# WhatsApp группы пользователя
curl -X GET "https://chatrassylka.ru/api/whatsapp/groups/{userId}" \
  -H "Authorization: Bearer {token}"

# Telegram группы пользователя  
curl -X GET "https://chatrassylka.ru/api/telegram/groups/{userId}" \
  -H "Authorization: Bearer {token}"
```

В ответе поле `is_selected: true` означает, что группа выбрана для рассылки.
