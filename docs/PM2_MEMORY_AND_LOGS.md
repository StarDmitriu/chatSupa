---

## Команды

- **Список и память:** `pm2 list`
- **Детали процесса:** `pm2 show frontend` / `pm2 show backend`
- **Логи в реальном времени:** `pm2 logs`
- **Только ошибки:** `pm2 logs --err`
- **Очистить логи:** `pm2 flush`
- **Рестарт после деплоя:** `pm2 restart frontend` или `pm2 restart all`

Файлы логов:
- `~/.pm2/logs/frontend-out-1.log`, `frontend-error-1.log`
- `~/.pm2/logs/backend-out-0.log`, `backend-error-0.log`

## Ошибка фронтенда: "Failed to find Server Action"

**Что в логах:**  
`Error: Failed to find Server Action "x". This request might be from an older or newer deployment.`

**Причина:** После нового билда Next.js меняет внутренние ID Server Actions. У части пользователей в кэше остаётся старый JS — он шлёт старый action ID, сервер его не находит → ошибка в логах и возможные рестарты/нестабильность.

**Что сделано:** В `ecosystem.config.cjs` для фронтенда добавлена переменная **`NEXT_SERVER_ACTIONS_ENCRYPTION_KEY`** (фиксированное значение). Один и тот же ключ используется после каждого деплоя, поэтому старые и новые билды согласованы по действиям.

После изменения конфига перезапустите фронт:
```bash
pm2 restart frontend
```

Пользователям при странном поведении после обновления сайта можно порекомендовать жёсткое обновление страницы: **Ctrl+Shift+R** (или Cmd+Shift+R).

## Ошибки бэкенда (логика, не падения)

В логах бэкенда часто встречаются ожидаемые ошибки Telegram API:

- **CHAT_ADMIN_REQUIRED** — в группе/канале нужны права админа для отправки.
- **CHANNEL_PRIVATE** / **CHANNEL_INVALID** — канал недоступен или неверный.
- **CHAT_WRITE_FORBIDDEN** — в чат писать запрещено.

Они фиксируются в БД (`last_send_error`) и отображаются в интерфейсе выбора групп. Рестарты backend (61 и т.п.) обычно связаны с деплоями или ручными рестартами, а не с этими ошибками.

## История рестартов

Счётчик рестартов (↺) в `pm2 list` увеличивается при каждом перезапуске процесса (вручную, из-за лимита памяти или падения). После применения фикса Server Actions число рестартов фронтенда от этой ошибки должно перестать расти.
