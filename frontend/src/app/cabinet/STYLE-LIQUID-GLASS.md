# Стиль Liquid Glass — единый визуальный стиль

Единый визуальный стиль карточек и кнопок: личный кабинет, «Ваши шаблоны», создание/редактирование шаблона, управление группами WhatsApp и Telegram, рассылки, аналитика, админ-панель. Все переменные задаются на корневом контейнере каждой страницы.

## Фон страницы

Страницы имеют **мягкий градиентный фон**: от белого сверху к светлому тёплому кремовому снизу. Секции по умолчанию без своего фона — контент на общем полотне. Карточки и панели — в стиле liquid glass (градиент, blur, рамка, тень, блики).

## Переменные (использовать в новых блоках)

| Переменная | Назначение |
|------------|------------|
| `--cabinet-block-gap` | Единый отступ между блоками (ЛК, по умолчанию 24px) |
| `--*-page-bg` | Фон страницы (мягкий линейный градиент: белый → светлый крем) |
| `--*-glass-bg` | Фон карточки (тёплый полупрозрачный градиент) |
| `--*-glass-border` | Рамка карточки |
| `--*-glass-shadow` | Тень карточки + inset-блик по верхнему краю |
| `--*-glass-inner-border` | Края внутренних блоков |
| `--*-glass-inner-shadow` | Тень внутреннего блока + inset-блик |
| `--*-glass-btn-bg` | Фон кнопок |
| `--*-glass-btn-border` | Рамка кнопок |
| `--*-glass-btn-shadow` | Тень кнопок |
| `--*-glass-btn-hover-bg` | Фон кнопок при наведении |
| `--*-glass-btn-hover-shadow` | Тень кнопок при наведении |

Префикс `*`: в ЛК — `--cabinet-*`, на странице шаблонов — `--tpl-*`, в создании/редактировании шаблона — `--tedit-*`, в управлении группами — `--grp-*`, на странице рассылок — `--camp-*`, на странице аналитики — `--ana-*`, в админке — `--adm-*`.

## Шапка (header) — единый паттерн

На всех перечисленных страницах шапка устроена одинаково:

- **Лейбл страницы** (например «Личный кабинет», «Ваши шаблоны», «Создание шаблона», «Управление группами WhatsApp») — мелкий uppercase, полупрозрачный серый, класс `*__label`.
- **Логотип** «ЧатРассылка» — ссылка на главную, класс `*__logo`.
- **Кнопки-таблетки** (pills) в навигации — liquid glass, при hover лёгкий подъём `translateY(-2px)`.
- **Фон шапки**: псевдоэлемент `::before` — тёплый градиент в тон страницы, без белой полосы.
- **Sticky**: шапка с `position: sticky; top: 0;`, чтобы не находилась внутри элемента с `transform` (иначе sticky ломается).

Классы шапки (общие): `.tpl-header`, `.tpl-header__container`, `.tpl-header__label`, `.tpl-header__row`, `.tpl-header__logo`, `.tpl-header__nav`, `.tpl-header__pill`. В ЛК используется `.cabinet-header` и `.cabinet-header__*` с теми же принципами.

## Правила для новых блоков

1. Фон секции: `background: var(--*-glass-bg);` + `border: var(--*-glass-border);` + тень.
2. Карточка: `background: var(--*-glass-bg);` + `backdrop-filter: blur(18px);` + `border` + `box-shadow` + `::before` с радиальными бликами.
3. Внутренняя панель: `border: var(--*-glass-inner-border);` + `box-shadow: var(--*-glass-inner-shadow);`.
4. Кнопки: переменные `--*-glass-btn-*`, при hover — `-hover-*`.
5. Контент над бликом: дочерним элементам задать `position: relative; z-index: 0;`.

## Интервал отправки (send_time) — где хранится и как ведёт себя

- **Хранение**: интервал хранится **один на группу** в таблицах `whatsapp_groups` и `telegram_groups` (поле `send_time`). Связки «шаблон — группа» отдельно не хранят интервал.
- **Поведение**: если одну и ту же группу добавить в два разных шаблона и в каждом редакторе шаблона поменять интервал — сохранится **последнее** сохранённое значение. Оно будет использоваться для этой группы во всех шаблонах и рассылках.
- **Вывод**: интервал в «Управлении группами» и в редакторе шаблона редактирует одно и то же поле. Убирать интервал из управления группами не нужно: это удобное место, чтобы задать интервал по умолчанию для группы один раз. В шаблонах можно дополнительно задать «общий интервал для всех выбранных групп этого шаблона» (применить ко всем выбранным) — тогда массово обновляется `send_time` у этих групп.

## Добавление новых страниц

При добавлении новой страницы в том же стиле:

1. Задать на корневом контейнере переменные (скопировать значения из `.tpl` или `.cabinet`).
2. Добавить шапку с лейблом страницы, логотипом и кнопками-таблетками.
3. Применять к карточкам и кнопкам переменные по правилам выше.
