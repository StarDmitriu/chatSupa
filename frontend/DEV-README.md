# Разработка frontend (Next.js)

## Стили не грузятся на localhost:58213

Порт **58213** — это порт превью Cursor (Simple Browser), а не dev-сервер Next.js. Стили и статика (`/_next/static/...`) отдаются с того же хоста, поэтому на 58213 они не подгружаются.

**Что делать:** открывать приложение на порту, где реально крутится dev-сервер.

### Запуск dev и просмотр

```bash
cd /var/www/frontend
npm run dev
```

Dev-сервер поднимается на **http://localhost:3001**. Открывайте в браузере:

- **http://localhost:3001** — стили и статика будут грузиться.

Не используйте для разработки **http://localhost:58213** — это другой порт, там нет вашего Next.js dev-сервера.

### Если нужен именно порт 58213

Можно запустить dev на 58213:

```bash
cd /var/www/frontend
npx next dev -p 58213
```

После этого открывайте **http://localhost:58213** — стили будут работать.
